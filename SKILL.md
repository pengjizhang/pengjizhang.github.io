# 博客发布 Skill

## 触发方式
说"发一篇关于XX的文章"或"发博客：XX"

## 自动流程

### 1. 写文章
- 根据主题写 Markdown，存到 `D:\code\blog-pengjizhang\文章名.md`
- 深度文章用纯文字，图解文章用 `<pre class="mermaid">` 嵌入 Mermaid
- 每篇开头必须有 frontmatter：`title: "标题"`

### 2. 更新目录
- 在 `index.md` 文章列表里加链接
- 格式：`- [标题](文件名)`

### 3. 自动提交
- `git add -A && git commit -m "Add 文章名" && git push`
- 30 秒后上线 `https://pengjizhang.github.io/文章名`

### 4. 发布后提醒
博客上线后，主动提醒：

> ✅ 博客已上线：[链接]
>
> 要不要同步到其他平台？
> - 🔵 **知乎**：直接复制博客正文，发一篇回答或文章
> - 🔴 **小红书**（适合图解类文章）：截 Mermaid 图 + 摘 3 句精华，发图文笔记

## 内容类型建议

| 类型 | 适合平台 |
|------|----------|
| 深度技术分析 | 博客 + 知乎 |
| 系统思维/方法论 | 博客 + 知乎 |
| 图解速览 | 博客 + 小红书 |
| 个人反思 | 博客 + 知乎（谨慎） |
| 国际标准/技术趋势 | 博客 + LinkedIn |

## 安全规则
- 不写公司内部信息
- 不贴未公开数据
- 个人观点用"我认为"，不引用公司口径
