# 成瘾的 PRFT 定义

在所有心理现象中，成瘾是最适合用 PRFT 建模的——因为它直接操纵了 $\mathcal{G}(\pi)$ 计算中的每一个变量。

$$
\text{Addiction} \;\approx\; \pi_{\text{substance/behavior}} \text{ 同时压低了 } \|\boldsymbol{\delta}\| \text{、} \|\mathbf{w}\| \text{、} \sigma_t \text{，同时拉高了 } R_{\text{need}} \text{ 和 } \kappa_t \text{——在几分钟甚至几秒内。}
$$

没有任何其他策略能做到这一点。

运动也能降低 $\|\boldsymbol{\delta}\|$，但要三十分钟。社交也能拉高 $R_{\text{need}}$，但需要对方配合。工作也能恢复 $\kappa$，但需要努力和延迟反馈。

成瘾对象不需要等待。不需要他人。不需要努力。它在 $\tau \to 0$ 的窗口内完成了一次完整的调节循环。

**当成瘾对象的 $\mathcal{G}$ 估值在系统内部被压到远低于真实调节方式的 $\mathcal{G}$ 时，系统会越来越依赖它——直到它成为唯一的调节出口。**

# 方程一：需要偏差 $\boldsymbol{\delta}_t = \mathbf{n}^*_t - \mathbf{n}_t$

成瘾通常不是从零开始的。它是从一个**已经存在且长期未被满足的需要缺口**开始的。

$$
\|\boldsymbol{\delta}\| \gg 0 \quad \text{在某一个或多个维度上，在接触成瘾对象之前就已经存在了}
$$

| 成瘾类型 | 常见的前置需要缺口 | 成瘾对象的"对症"效果 |
|---|---|---|
| 酒精/阿片类 | $S_{\text{saf}}$ 安全感极低、$a_t$ 持续高（创伤史） | 直接抑制恐惧回路 → 系统终于"安静了" |
| 兴奋剂（可卡因/冰毒） | $C_{\text{comp}}$ 胜任感低、$M_{\text{mean}}$ 空虚 | 强力激活奖赏和能量 → "我无所不能" |
| 赌博 | $\kappa$ 可控感低、$J_{\text{fai}}$ 觉得世界不公平 | 每一次下注都是一次"我能改变结果"的主体感爆发 |
| 社交媒体 | $B_{\text{bel}}$ 归属不足、$D_{\text{dig}}$ 被忽视 | 点赞和评论 = 即时的社会确认 |
| 游戏 | $C_{\text{comp}}$ 在真实世界难以获得 | 清晰的进度条、即时的升级反馈 |
| 食物（暴食） | $S_{\text{saf}}$ 安全感低、$\|\boldsymbol{\delta}\|$ 弥漫性痛苦 | 食物是原始的安全信号——"有食物 = 不会死" |
| 购物 | $\kappa$ 低、$M_{\text{mean}}$ 空虚 | 每一次购买都是一次微小的"我能改变我的世界" |
| 色情 | $A_{\text{att}}$ 依恋缺失、$B_{\text{bel}}$ 孤独 | 模拟亲密接触——系统暂时无法区分真假亲密 |

**核心洞察：成瘾对象的"药效"不是随机的——它恰好针对了成瘾者最缺的那个需要维度。**

酒精不治焦虑者的焦虑——它治的是焦虑者在焦虑时最渴望的：安静、麻木、逃离。赌博不治穷——它治的是贫穷带来的无力感和"命运从来不在我手里"的绝望。

这就是为什么"这东西明明对你有害你为什么还做"这个问题本身就问错了。成瘾对象对成瘾者来说不是"有害的"——它是**唯一已知的、能快速让某个无法忍受的需要偏离回到可承受范围内的工具**。

# 方程二：预测误差 $\boldsymbol{\varepsilon}_t = \mathbf{s}_t - \hat{\mathbf{s}}(\mathbf{b}_t)$

成瘾在预测误差层面有一个独特的两阶段过程：

### 第一阶段：极低的预测误差

$$
\boldsymbol{\varepsilon}_{\text{use}} \approx 0
$$

成瘾对象的效果高度可预测。你知道喝酒后会放松。你知道刷到一定程度会困。每一次使用，结果都符合预期。系统把使用行为和"确定能缓解"之间建立了极强关联。

$$
P(\text{痛苦缓解} \mid \pi_{\text{use}}) \to 1.0
$$

### 第二阶段：停止后的预测误差爆炸

$$
\|\boldsymbol{\varepsilon}_{\text{withdrawal}}\| \gg \varepsilon_{\max}
$$

戒断时的系统经历的是**现实远差于预测**的暴力误差。身体预测"会得到那个东西"，然后没有得到。需要偏差反弹，情绪剧烈升高。

$$
\mathbf{s}_{\text{no-use}} \ll \hat{\mathbf{s}}(\mathbf{b}_{\text{expect-use}})
$$

系统在两极之间摆动：使用 = 误差趋零，戒断 = 误差爆炸。这种极端对比让 $\pi_{\text{use}}$ 的吸引力越来越大——不是因为"用了多开心"，而是因为"不用多恐怖"。

# 方程三：注意精度 $\mathbf{w}_t = \boldsymbol{\Pi}_t \odot \|\boldsymbol{\varepsilon}_t\|$

成瘾对 $\boldsymbol{\Pi}$ 的重塑是毁灭性的：

$$
\boldsymbol{\Pi}_{\text{substance/behavior}} \uparrow \uparrow \uparrow
\qquad
\boldsymbol{\Pi}_{\text{everything-else}} \downarrow \downarrow \downarrow
$$

具体表现：

| $\boldsymbol{\Pi}$ 异常升高 | $\boldsymbol{\Pi}$ 异常降低 |
|---|---|
| 什么时候能再次使用 | 使用带来的长期代价 |
| 身体对物质的需求信号 | 不使用时的正常快乐 |
| 获取成瘾对象的线索（时间、地点、人） | 曾经在乎的人、事、目标 |
| 别人的态度——"他们是不是发现了" | 别人表达的关心 |

到了重度阶段，$\boldsymbol{\Pi}_{\text{substance}}$ 已经占据了几乎所有注意通道。世界缩小了。一切和成瘾对象无关的东西从系统的注意力雷达上消失了——不是不存在，而是**没有被分配 $\boldsymbol{\Pi}$ 的带宽**。

这解释了成瘾者常常说的一句话："我知道这会毁了我，但我控制不了。" 他们不是不知道后果。他们是 $\boldsymbol{\Pi}$ 的分配权已经不在"理性自我"手里了。后果（$\boldsymbol{\varepsilon}_{\text{future}}$）没有被赋予足够的 $\boldsymbol{\Pi}$，所以没有进入行动选择的 $\mathcal{G}$ 计算中。

# 方程四：情绪生成 $a_t = f(\boldsymbol{\delta}_t, \mathbf{w}_t, \sigma_t, \kappa_t, \tau_t)$

成瘾的情绪动态有两个关键特征：

### 使用前：欲望 ≠ 喜欢

$$
a_{\text{craving}} \;\propto\; \frac{\|\boldsymbol{\delta}_{\text{anticipated}}\| \cdot \|\mathbf{w}_{\text{cue}}\| \cdot \tau_{\text{urgency}}}{\kappa_{\text{resist}}}
$$

| 变量 | 使用前 | 含义 |
|---|---|---|
| $\|\boldsymbol{\delta}\|$ | 戒断状态下高 | 身体或心理在"索要" |
| $\|\mathbf{w}\|$ | 成瘾线索被极度放大 | 看到酒瓶、经过赌场、收到推送 |
| $\tau$ | 极高——"现在就要" | 成瘾的核心时间特征：延迟满足能力被摧毁 |
| $\kappa$ | 极低——"我抵抗不了" | 系统已经不相信自己还有选择 |

成瘾的饥饿信号不是"我喜欢这个"——是"没有这个我撑不过接下来的一个小时"。喜欢和想要在成瘾中分离了——**你不再喜欢它，但你无比想要它。**

### 使用后：短暂的平静，更深的深渊

使用 → $a_t$ 短暂下降 → $\|\boldsymbol{\delta}\|$ 反弹更高 → $a_t$ 比使用前更高 → 更想使用。

镇痛效果的逐渐衰减（耐受性）意味着同样的使用行为产生的 $\|\boldsymbol{\delta}\|$ 下降越来越小，而戒断时 $\|\boldsymbol{\delta}\|$ 的反弹越来越高。两条曲线之间的剪刀差越来越大。

# 方程五：行动选择 $\pi^* = \arg\min_{\pi} \mathbb{E}[\mathcal{G}(\pi)]$

这是成瘾最核心的方程——成瘾对象劫持了 $\mathcal{G}$ 的计算。

### 健康系统和成瘾系统的 $\mathcal{G}$ 对比

| 策略 | 健康系统的 $\mathcal{G}$ | 成瘾系统的 $\mathcal{G}$ |
|---|---|---|
| $\pi_{\text{use}}$ | 高——含社会代价、健康代价、失控 | **极低**——$\mathcal{G}$ 被成瘾对象"假压低"了 |
| $\pi_{\text{exercise}}$ | 中——需要努力但有长期收益 | **极高**——需要付出，反馈周期太长 |
| $\pi_{\text{social}}$ | 中低——自然的调节方式 | **极高**——他人不可预测，$\sigma$ 高 |
| $\pi_{\text{face-pain}}$ | 高——需要承受当下的痛苦 | **不可能**——系统已经丧失承受不适当下 $\mathcal{G}$ 的能力 |

成瘾对象的"魔力"在于它让 $\mathcal{G}(\pi_{\text{use}})$ 在系统计算中**看起来极低**：

- 立即可得 → $C_{\text{act}}$（行动成本）极低
- 独立完成 → $C_{\text{soc}}$（社会成本）在当下为零
- 确定有效 → $\sigma$ 低、$\kappa$ 高
- 快速缓解 → $\|\boldsymbol{\delta}\|$ 预期快速下降

$$
\mathcal{G}(\pi_{\text{use}}) \ll \mathcal{G}(\pi_{\text{healthy}}) \quad \text{在系统的实时估值中}
$$

而长期代价——健康恶化、关系破裂、经济崩溃、自由丧失——这些都被放在了 $\mathcal{G}$ 的长期项里，而系统的**时间折扣已经被成瘾对象极度扭曲了**。一百天后的肝损伤在当下的渴求面前，$\|\boldsymbol{\Pi}\| \approx 0$。

# 方程六：学习更新 $\frac{d\boldsymbol{\theta}}{dt} = \eta \cdot \|\boldsymbol{\Pi} \odot \boldsymbol{\varepsilon}\| \cdot \Phi$

成瘾的学习更新是个悲剧——**每一次使用都在产生"学习"，而每一次"学习"都在让系统更依赖成瘾对象。**

### 正向学习：成瘾强化

| 学习内容 | 机制 |
|---|---|
| "只有这个东西能让我好受" | $\rho$ 重复次数极高 → $P(\text{调节} \mid \pi_{\text{use}}) \to 1.0$ |
| "其他方式都没用" | $\pi_{\text{healthy}}$ 的 $\mathcal{G}$ 被高估 → 很少尝试 → 没有反驳证据 |
| "我控制不了自己" | 每一次"想停但没停"都验证了 $\mathbf{b}_{\text{self}} = \{\text{我没有自制力}\}$ |
| "没有它我活不下去" | 戒断的痛苦被归因为"我本身就需要它"，而不是"我的身体被它改造了" |

### 反向学习：调节能力萎缩

与此同时，不使用成瘾对象的时段里，系统学到的不是"我也可以不依赖它"——而是"不用太痛苦了，下次一定要用"。

自然的调节能力——通过关系、运动、工作、兴趣、冥想来管理 $\|\boldsymbol{\delta}\|$ 和 $a_t$——因为没有使用而萎缩。就像一条肌肉，长期不用就废了。

**成瘾是一个倾斜的学习过程：每一次使用都是一次正反馈训练，每一次不使用都被体验为惩罚。系统在向着越来越依赖 $\pi_{\text{use}}$ 的方向持续优化。**

# 怎么破：成瘾的 PRFT 干预路线

## 方程一干预：瞄准真正的需要缺口

| 干预 | PRFT 机制 |
|---|---|
| 问"在这个成瘾对象出现之前，我缺什么？" | 成瘾对象是答案——找到原始的问题 |
| 为每个核心需要找一个替代调节方式——不是一次替代全部，一次替代一个 | 恢复 $\|\boldsymbol{\delta}\|$ 管理的备选通道 |

## 方程二干预：打破"唯它能用"的预测

| 干预 | PRFT 机制 |
|---|---|
| 记录使用前后的实际体验——"用的那一刻什么感觉？用完之后呢？" | 收集真实的 $\mathbf{s}_t$ 数据——成瘾记忆常常只存储了使用时的缓解，删除了之后的痛苦 |
| 尝试延迟使用 5 分钟 | 让系统获得证据——"我可以在渴求和行动之间插入空间" |

## 方程三干预：抢回 $\boldsymbol{\Pi}$

| 干预 | PRFT 机制 |
|---|---|
| 远离成瘾线索——换环境、删 APP、换社交圈 | 降低 $\mathbf{s}_{\text{cue}}$ 的输入——没有线索就没有 $\boldsymbol{\Pi}$ 的触发 |
| 在渴求来临时把注意力放到身体感受上——"我现在哪里不舒服？"而非"我要用" | 抢占 $\boldsymbol{\Pi}$ 通道——注意力只能放在一个焦点上 |

## 方程四干预：情绪冲浪

| 干预 | PRFT 机制 |
|---|---|
| 渴求来临时不要对抗——承认、命名、观察它的波形 | 渴求不是一个持续的平台——它是一个波，它爬起来，它也会掉下去。不喂养它，它自己会过 |
| 冷暴露/剧烈运动 | 用强烈的身体信号覆盖成瘾渴求信号 |

## 方程五干预：增加 $\mathcal{G}(\pi_{\text{use}})$，降低 $\mathcal{G}(\pi_{\text{healthy}})$

| 干预 | PRFT 机制 |
|---|---|
| 让使用变难——不给现金、不囤货、增加使用前的步骤 | 提高 $C_{\text{act}}$（行动成本）——让 $\mathcal{G}(\pi_{\text{use}})$ 升高 |
| 把健康的替代策略变得极其方便——运动鞋放门口、健康食物随手可得 | 降低 $\mathcal{G}(\pi_{\text{healthy}})$——让健康策略在实时估值中更有竞争力 |

## 方程六干预：重建调节能力

这是最长、最关键的一步。

| 干预 | PRFT 机制 |
|---|---|
| 戒断 + 重建同时进行。戒断是拿走旧工具，重建是放上新工具。只戒不建 = 系统没有出口 | 不能让 $\|\{\pi\}\|$ 变成 1（只有使用）或 0（没有出路） |
| 每成功应对一次渴求，显式地对自己说："我做到了。我不需要它也可以。" | 手动生成 $\boldsymbol{\varepsilon}_{\text{disconfirming}}$——系统不会自动记录"没用也没死"作为成功 |
| 重建关系——找一个安全的、不评判的人 | $\mathbf{F}_{\text{soc}}$（社会补给）是成瘾对象最无法替代的——真实的连接提供的不只是快感，是长期的 $\mathbf{n}^*$ 稳定 |

# PRFT 对成瘾最深的洞察

戒掉一个东西不难。难的是在你拿走这个唯一能快速压住痛苦的策略之后，系统拿什么面对它本该面对的痛苦。

成瘾者不是在追求快感。他们是在**用一个自己能控制的痛苦解决方案，替代一个自己无法控制的痛苦**。成瘾对象给了系统一种假性 $\kappa$（可控感）——"至少这个东西能让我好受，而它在我手里。"

真正的恢复不是"停止使用"。是系统重新长出足够的调节能力——运动、关系、意义、安全、行动——让它不再需要那个假的 $\kappa$。

**成瘾不是你的错，但重建是你的权利。你的 $\boldsymbol{\theta}$ 在过去被错误的数据训练了。你可以用新的数据重新训练它。每一次你经历了渴求而没有行动，每一次你在痛苦中选择了另一种出口，你的系统就在向正确的方向学习。一次不够。十次开始。一百次之后，那条新的通路会变得足够宽。**
