# 视角是什么：一张自我加固的失真地图

我们常用"视角"这个词，但很少停下来问：它到底是什么？这篇文章用三种思维框架——第一性原理、苏格拉底追问、心智模型组合——层层剥开"视角"的本质。

## 第一性原理：视角是一个过滤器

拆到最底层，视角由三样东西决定：

**1. 你站在哪** — 位置决定什么信息能到你面前。CEO 看到汇总数据，一线看到具体案例。两个人都在系统里，但被位置塞了不同的输入。

**2. 你想干嘛** — 目的决定你看什么、忽略什么。想赢的人看机会，想不输的人看风险。同一个信息，目的不同，归为"信号"还是"噪音"的判定完全不同。

**3. 你脑子里有什么** — 经验、信念、语言决定了你能不能认出你看到的东西。你没有那个模型，东西站在你面前你也看不见。

视角不是你可以随便选的"观点"。这三个因素交汇后**自动形成**了你的视角——大部分工作在你意识不到的情况下就已经完成了。

所以：**视角 = 你在看什么 + 你忽略什么。** 它是信息的过滤器，过滤器形状由位置、目的、心智模型共同决定。

## 苏格拉底追问：视角怎么形成？

追问"视角怎么形成"，走到一个关键地方：

视角的形成 = **日积月累 + 关键事件**，二者分不开。

积累是土壤，事件是种子。没有土壤，种子不会发芽。但追问"土壤最早什么时候有的"，发现**没有明确起点**。

这意味着："换一个视角"可能是个错觉。你不是换了一个新视角，你只是在旧的土壤里加了一层新东西。视角没有"出厂重置"这回事。

追问到最后，只能说"不知道"——这不是失败，是探到了边界。

## 心智模型组合：三个模型拼图

单个模型会漏东西。用三个模型并行看，互相补盲区。

### 地图-领土：视角是地图，不是领土

视角就是你手里那张地图。地图一定在三个地方失真：**简化**（细节丢了）、**扭曲**（比例不对）、**遗漏**（有些东西压根没画进去）。

你拿着一份地图，不能说"这就是土地本身"。同理，你的视角不等于现实——它只是现实的一个**特定比例的压缩**。投影永远有损失，但你意识不到损失了什么，因为你看不到被滤掉的部分。

### 反馈回路：视角是自我锁定的系统

```
你相信什么 → 你注意什么 → 你找到什么证据 → 你更相信什么
     ↑                                              ↓
     └──────────────── 强化回路 ────────────────────┘
```

视角不是被动接收信息，是**主动过滤信息**。滤进来的东西，又反过来加固那个过滤器本身。

这就是换视角很难的原因——不是因为"固执"，是因为回路在自转。你每次都"看到了证据"。视角会自动排除那些能推翻它的信息。

### 系统思维：视角是位置帮你选的

一个系统里，不同节点收到的信息**天生不对称**。位置在你不选择的情况下，已经替你选了你能看到什么。

视角不完全是你选的。一半是你的选择，一半是你的位置强塞给你的。

## 合成：三股力量同时作用

视角难换，因为三股力量叠加：

1. **地图本来就有损失** — 天然失真
2. **回路一直自转** — 自我锁定
3. **位置帮你过滤** — 你不选也得选

这三股力量加在一起，你看到的不是"现实的一个版本"——你看到的是**一个被位置过滤过的、自我加固的、失真的投影**，而你意识不到它是投影。

所以：

> **视角 = 你在系统里被位置塞了一张地图，然后这张地图会自我加固，让你以为它就是领土本身。**

要改变视角，至少得打断其中一个环节。
